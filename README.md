# 🏗️ Poised Project Management System

A Java-based project management system developed for a structural engineering company called **Poised**.
The application helps manage construction projects, track project details, monitor deadlines, and maintain customer, contractor, and architect information.

## 🚀 Features

### 📌 Project Management

* Create new projects
* Update existing project information
* Search projects by:

  * Project Number
  * Project Name
* Automatically generate a project name using the customer's surname if no project name is provided

### ✅ Project Finalisation

* Mark projects as completed/finalised
* Record project completion dates

### 📅 Deadline Tracking

* View projects that are still incomplete
* View overdue projects past their due dates

### 👥 Stakeholder Management

Store and manage information for:

* Customers
* Contractors
* Architects

Including:

* Name
* Telephone number
* Email address
* Physical address

## 🏗️ System Overview

The system allows a structural engineering company to efficiently manage multiple building projects such as:

* Houses
* Apartment blocks
* Stores
* Other structural developments

Each project stores:

* Project details
* Financial information
* Deadlines
* Stakeholder contact information
* Completion status

## 📂 Example Functionalities

### ➕ Add New Project

Capture:

* Project number
* Building type
* Address
* ERF number
* Fees
* Amount paid
* Deadline
* Customer/Contractor/Architect details

### ✏️ Update Existing Project

Modify:

* Deadlines
* Fees
* Contact details
* Payment information

### 🔍 Search Project

Find projects quickly by:

* Project name
* Project number

### ⏰ View Overdue Projects

Display projects that have passed their due date but are not yet finalised.
